The screen refresh gesture works when the function is turned off

App: FossifyOrg/File-Manager
Version: 1.0.1
Issue: https://github.com/FossifyOrg/File-Manager/issues/136
APK: https://github.com/FossifyOrg/File-Manager/releases/download/1.0.1/file-manager-2-foss-release.apk
APK Tag: 1.0.1
Gestures: pull-to-refresh
Primary Gesture: swipe

Steps to Reproduce:
  1. Go to "Settings".
  2. Scroll to "Scrolling"
  3. Off "Enable pull-to-refresh from the top"
  4. Select folder.
  5. Cancel selection.
  6. Select folder again.

Expected Behavior:
  Normal selection of an object.

Actual Behavior:
  Page refresh effect (arrow in white circle).
