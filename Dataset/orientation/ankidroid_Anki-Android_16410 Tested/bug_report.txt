[BUG]: Changing screen orientation clears stats' search options

App: ankidroid/Anki-Android
Version: 2.18.0
Issue: https://github.com/ankidroid/Anki-Android/issues/16410
APK: https://github.com/ankidroid/Anki-Android/releases/download/v2.18.0/AnkiDroid-2.18.0-amazon-universal.apk
APK Tag: v2.18.0
Gestures: rotate, orientation, landscape
Primary Gesture: orientation

Steps to Reproduce:
  1. Go to statistics
  2. Search something in the search bar. Or select collection over deck/all history over last twelve months
  3. Change screen orientation. Screen would rotate.

Expected Behavior:
  Only screen orientation would change

Actual Behavior:
  The search options sets back to default.The screen loads once again.
