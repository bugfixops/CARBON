Eraser size not relative to zoom on minimum size

App: FossifyOrg/Paint
Version: 1.0.0
Issue: https://github.com/FossifyOrg/Paint/issues/25
APK: https://github.com/FossifyOrg/Paint/releases/download/1.0.0/paint-1-foss-release.apk
APK Tag: 1.0.0
Gestures: zoom in, zoom

Steps to Reproduce:
  1. Draw a line
  2. Set brush size to the minimum
  3. Run eraser through line
  4. Zoom in/out
  5. Run eraser through line at different position

Expected Behavior:
  Eraser size should change relative to the zoom.

Actual Behavior:
  Eraser size doesn't change.
