[BUG]: IO Cards Go to the Wrong Deck

App: ankidroid/Anki-Android
Version: 2.20alpha1
Issue: https://github.com/ankidroid/Anki-Android/issues/17393
APK: https://github.com/ankidroid/Anki-Android/releases/download/v2.20alpha1/AnkiDroid-2.20alpha1-amazon-universal.apk
APK Tag: v2.20alpha1
Gestures: double-click
Primary Gesture: double_tap

Steps to Reproduce:
  1. Go to **Settings > General**.
  2. Click **Deck for new cards** and select `use current deck`.
  3. Go back to deck picker and select a deck.
  4. Now, double-click **＋** to open the Add screen.
  5. Select the IO note type.
  6. Change deck. Now you've selected a different deck and cards should go here.
  7. Create a IO card from here.

Expected Behavior:
  The deck where the IO cards created go should be the one we selected at #6 but the cards go to the previously selected deck.

Actual Behavior:
  (see full report below)
