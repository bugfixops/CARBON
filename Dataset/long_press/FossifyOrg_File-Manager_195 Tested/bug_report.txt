Unnecessary refresh of ZIP file icons when closing bottom sheets

App: FossifyOrg/File-Manager
Version: 1.1.0
Issue: https://github.com/FossifyOrg/File-Manager/issues/195
APK: https://github.com/FossifyOrg/File-Manager/releases/download/1.1.0/file-manager-3-foss-release.apk
APK Tag: 1.1.0
Gestures: long-press, click, press

Steps to Reproduce:
  1. Open a folder in the _Files_ tab that contains at least one ZIP file
  2. Long-press on a ZIP file
  3. Click the **Open with** icon in the action menu
  4. Close the bottom sheet dialog

Expected Behavior:
  ZIP file icons should remain unchanged (no visual refresh).

Actual Behavior:
  ZIP file icons unnecessarily refresh when the bottom sheet is closed.
