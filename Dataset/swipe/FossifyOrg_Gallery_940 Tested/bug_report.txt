Disabled notch overlaps brightness control area

App: FossifyOrg/Gallery
Version: 1.12.0
Issue: https://github.com/FossifyOrg/Gallery/issues/940
APK: https://github.com/FossifyOrg/Gallery/releases/download/1.13.1/gallery-28-foss-release.apk
APK Tag: 1.13.1
Gestures: swipe, brightness gesture, volume gesture

Steps to Reproduce:
  1. Disable "Show notch if available" in settings
  2. Enable "Allow controlling photo brightness with vertical gestures"
  3. Rotate device horizontally so that the notch area is on the left
  4. Open an image
  5. Try to change brightness using vertical gestures on the left side of the screen

Expected Behavior:
  The vertical gesture for changing brightness works the same as in vertical screen orientation.

Actual Behavior:
  The brightness gesture area gets overlapped by the "hidden notch" area, making it almost impossible to change the brightness. It works only occasionally, probably because there are still a few pixels not overlapped by the notch area.

--- Full Issue Body ---
Apparently, the brightness gesture area gets overlapped by the "hidden notch" area, making it almost impossible to change the brightness.
