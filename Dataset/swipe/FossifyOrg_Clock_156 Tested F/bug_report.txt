Timer alarm turned off by swiping from the status bar

App: FossifyOrg/Clock
Version: 1.2.0
Issue: https://github.com/FossifyOrg/Clock/issues/156
APK: https://github.com/FossifyOrg/Clock/releases/download/1.2.0/clock-3-foss-release.apk
APK Tag: 1.2.0
Gestures: notification, status bar
Primary Gesture: unknown

Steps to Reproduce:
  1. Set up any timer and start it
  2. Wait for the alarm sound
  3. Pull down from the status bar
  4. You'll not be able to silence the timer alarm by pushing the button from the notification. Instead touching the status bar area is sufficient to stop the alarm.

Expected Behavior:
  Pull down from the status bar to push the stop button, alarm sound stops and timer is reset.

Actual Behavior:
  Timer is not reset and needs manual intervention
