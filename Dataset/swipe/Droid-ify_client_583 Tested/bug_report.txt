[BUG] Swiping images zooms instead of zooming

App: Droid-ify/client
Version: 0.5.9
Issue: https://github.com/Droid-ify/client/issues/583
APK: https://github.com/Droid-ify/client/releases/download/v0.5.9.5/app-release.apk
APK Tag: v0.5.9.5
Gestures: zoom, image zoom
Primary Gesture: pinch_zoom

Describe the bug
Some (not all, so perhaps tied to uploaded image type) images on the app summary page, when the screen is touched to swipe, zoom the current image rather than swiping to the next image. I was careful to ensure nothing is interfering with my screen before submission of this issue.

To Reproduce
Steps to reproduce the behavior:

Go to https://www.f-droid.org/packages/io.github.muntashirakon.Music/
Click on any preview image
Touch the screen to swipe to the next image
See image zoomed rather than swiped
Expected behavior
Next image should be displayed instead of zooming.
